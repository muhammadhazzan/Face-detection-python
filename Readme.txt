👤 Face Recognition using Python

This is a beginner-friendly face recognition system built using Python and OpenCV. It detects and recognizes human faces from webcam input or image files.

🧠 Features
- Face detection using Haar Cascades
- Real-time recognition through webcam
- Custom training using labeled face images
- Lightweight and easy to set up

🛠️ Technologies Used
- Python 3
- OpenCV
- NumPy
- Haar Cascade Classifier
